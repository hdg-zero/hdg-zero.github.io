## Le projet

Ce dashboard a été conçu pour permettre aux équipes de visualiser leurs KPIs en temps réel. L'objectif était de créer une interface intuitive et performante.

## Stack technique

- **Frontend** : Next.js 14 avec App Router
- **Styling** : Tailwind CSS avec des composants custom
- **Backend** : Supabase pour la base de données et l'authentification
- **Charts** : Recharts pour les visualisations

## Fonctionnalités principales

- Temps réel via WebSockets
- Authentification sécurisée avec 2FA
- Graphiques interactifs et exportables
- Mode sombre natif
- PWA installable

## Défis rencontrés

Le plus grand défi a été d'optimiser les performances avec de gros volumes de données. J'ai implémenté un système de pagination virtuelle et de mise en cache intelligente.

## Résultats

- Temps de chargement initial : < 1.5s
- Score Lighthouse : 98/100
- Satisfaction utilisateur : 4.8/5
